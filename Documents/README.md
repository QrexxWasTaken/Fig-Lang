# README
## Fig Programming Language
### How to use Fig
1. Fork this repl
2. Change [main.fig](https://replit.com/@JosephHeinz/Fig-Lang#main.fig)
3. Run the repl

### Syntax
Write to the console: `sys.out('Message');`

Take input: `sys.in('Message');`

Wait: `sys.wait(time);`

Import a Package: `from system import [package];` or
`import [package];`
